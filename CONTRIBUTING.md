# Contributing

Thank you for helping improve PTX Studio.

This repository is customer-facing and is not an open-source software repository. PTX Studio application source code, private infrastructure, and release systems are not published here.

## What To Open

Use issues for:

- Bug reports
- Download problems
- Product suggestions
- Documentation corrections
- Release-note questions

## Pull Requests

Pull requests may be used for customer-facing documentation corrections or approved collaborator work.

Please do not submit:

- Application source code
- Decompiled or reverse-engineered code
- Server, hosting, storage, or deployment instructions
- Binaries or installers
- License keys, credentials, tokens, or private customer data

Submitting a pull request does not grant permission to use PTX Studio trademarks, application code, private assets, or proprietary systems.
