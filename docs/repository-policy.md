# Repository Policy

This repository is safe to make public only if it remains customer-facing.

## Allowed

- Public README, support, security, and changelog files
- Issue templates
- Release notes
- General product trust and signing information

## Not Allowed

- PTX Studio application source code
- Built executables or unpacked application directories
- Download mirror contents or manifests
- Server, storage, bucket, DNS, hosting, or deployment details
- Upload scripts, deployment scripts, or infrastructure configuration
- Credentials, API tokens, signing certificates, or passwords
- License files or customer/user data
- Local logs

## Internal Work Boundary

Builds, signing, uploads, hosting, and server operations belong in private systems, not in this customer-facing repository.
