# Changelog

All notable PTX Studio release changes should be documented here.

## [1.1.0](releases/v1.1.0.md)

- Major refinement release focused on performance, stability, rendering quality, and expanded professional editing tools across the PTX editor family.
- PTX Editor Pro adds Brush Builder, Modifiers tools, LUT/color grading support, and performance refinements.
- PTX Editor Express receives rendering, responsiveness, optimization, and UI polish updates.
- PTX Studio improves suite management foundations for installation, updating, and app handoff.
