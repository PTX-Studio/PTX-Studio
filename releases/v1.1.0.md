# PTX Studio 1.1.0

Release date: April 2026

PTX Studio Suite v1.1.0 is a major refinement release focused on performance, stability, rendering quality, and expanded professional editing tools across the PTX editor family.

## Products

- PTX Studio 1.1.0
- PTX Editor Express 1.1.0
- PTX Editor Pro 1.1.0

## Distribution

This release uses portable standalone Windows applications.

## PTX Editor Pro

- Dramatic performance improvements across the editing pipeline for smoother interaction, faster response, and improved workflow stability.
- Added a new Brush Builder system for more flexible custom brush creation and editing control.
- Added the Modifiers modal, including Liquify, Mesh Warp, and Distortion tools for direct creative manipulation.
- Added LUT and color grading support for stronger finishing, look development, and image styling workflows.
- General stability improvements and performance-focused refinements throughout the application.

## PTX Editor Express

- Updated rendering engine for improved responsiveness and a more reliable editing experience.
- General performance fixes and internal optimizations for smoother everyday use.
- Updated UI polish to better align Express with the broader PTX Studio visual system.

## PTX Studio

- Improved suite management foundations for installation, updating, and application handoff inside the PTX ecosystem.

## Bug Fixes And Improvements

- Resolved stability issues, improved rendering consistency, and refined core editor behavior across supported workflows.
