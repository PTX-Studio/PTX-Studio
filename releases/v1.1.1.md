# PTX Studio 1.1.1

Release date: April 2026

PTX Studio Suite v1.1.1 is a Quality of life refinement release focused on performance, stability, master export system for both PTX Editor Pro, and PTX Editor Express

## Products

* PTX Studio 1.1.0
* PTX Editor Express 1.1.1
* PTX Editor Pro 1.1.1

## Distribution

This release uses portable standalone Windows applications.

## PTX Editor Pro

* Expanded Master Export with per-target Profile Size and Document Size output modesAdded a new Brush Builder system for more flexible custom brush creation and editing control.
* Added Document Size export support so targets can preserve original PTX canvas dimensions and layer placement.
* Added per-target DPI controls for Master Export outputs, allowing different targets to carry different resolution metadata.
* Replaced manual format entry workflows with clear format dropdown controls for supported export types.

## PTX Editor Express

* Brought the streamlined Master Export workflow to Express with target import, profile assignment, format dropdowns, and document-size output.
* Added per-target DPI metadata controls for exported PNG, JPG, and WebP batches.

## PTX Studio

* Improved license handling

## Bug Fixes And Improvements

* Corrected export filename extension handling so saved files match the selected output format.
* Improved output folder handling for Master Export batch jobs.
* Updated standard export defaults to 92 DPI while preserving the 300 DPI print preset in PTX Editor Pro.
