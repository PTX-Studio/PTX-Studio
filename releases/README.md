# Releases

This folder holds customer-facing PTX Studio release notes.

Use one file per public version:

- `v1.1.0.md`
- `v1.2.0.md`
- `v1.2.1.md`

Do not commit application binaries, server manifests, private download paths, signing certificates, credentials, or deployment instructions here.

For downloadable files, use the official PTX Studio website or GitHub's built-in Releases page.
